The Box Model

Every HTML element is rendered in the browser as a box with a certain width and height. Each element's box is made up of a series of smaller nested boxes.

You can show the box model in the Chrome Inspector.

The **content box** is where text, images, and other elements will appear. This is the innermost box and can be sized using the `width` and `height` properties.

The **padding box** is the blank space between the content box and the border box, and can give individual elements breathing room next to the rest of the content on the page. Padding can be adjusted using the `padding` properties.

The **border box** is the region that marks the visual edge of the content box. The `background-color` property of the content box, for example, will not extend beyond the border. The border's width, color, and style can be adjusted all at once using the single `border` shorthand property, or individually using `border-width`, `border-style`, and `border-color`.

The **margin box** is the blank space between the border box of the element and the next HTML element. Margins are similar to padding but are positioned outside of the visual area of the element. Margins can be styled using the `margin` properties.

Block vs. Inline content.

HTML elements have natural behavior in the page ("normal flow.")

Block elements occupy the entire horizontal space, such as a section of a page or container.

Inline elements occupy a part of the horizontal space equal to the width needed to fit the inner content, such as a string of text, a link, or an image

You can change the natural behavior of an element by setting the display property.

Floating elements

The float CSS property lets us position an element to the left or right of its parent element. When we float an element, it shifts to the left or right until it touches the edge of its parent, or another floated element.

Centering Elements

To center an element on a page, or inside its containing element, you can set its left-margin and right-margin to auto.


                    .centered {
                      margin: 0 auto;
                    }

Demo this in the browser. Review how the values work for the margin property. Pay special attention to the idea of a "containing element".
